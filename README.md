# multi_chest_xray

This repository contains code and resources for the multi-class chest X-ray project.

Files included:
- `app.py` : main application
- `requirements.txt` : Python dependencies
- `best_model_v2 (1).pth` : trained model file

Quick start (local):

1. Create and activate a virtual environment:

```powershell
python -m venv venv
.\venv\Scripts\Activate.ps1
```

2. Install dependencies:

```powershell
pip install -r requirements.txt
```

3. Run the app:

```powershell
python app.py
```
